# Loader

The `Loader` component will provide the user with feedback that a portion of the
interface is fetching new data or content. It will keep the user informed of
system status as well as help the user avoid mistakes by preventing the user
from interacting with this area of the interface until the new data or content
is ready.

```ts
import { Loader } from "@jobber/components/Loader";
```

### Determinate

<Playground>
  {() => {
    const [isLoading, setIsLoading] = useState(false);
    const [currentValue, setCurrentValue] = useState(0);
    return (
      <Content>
        <Button
          label="Toggle Loading"
          onClick={() => setIsLoading(!isLoading)}
        />
        <Button
          label="Increase Loading Progress"
          disabled={!isLoading}
          onClick={() => setCurrentValue(Math.min(currentValue + 10, 100))}
        />
        <Button
          label="Decrease Loading Progress"
          disabled={!isLoading}
          onClick={() => setCurrentValue(Math.max(currentValue - 10, 0))}
        />
        <Loader
          determinate
          loading={isLoading}
          currentValue={currentValue}
          maxValue={100}
        >
          <Card title="Loading Content">
            <Content>
              <Text>
                Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do
                eiusmod tempor incididunt ut labore et dolore magna aliqua.
              </Text>
            </Content>
          </Card>
        </Loader>
      </Content>
    );
  }}
</Playground>
